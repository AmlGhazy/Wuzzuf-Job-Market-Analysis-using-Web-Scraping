# 🔍 Wuzzuf Job Market Analysis using Web Scraping

<img align="right" width="320" src="images/overview.png">

---

## 📊 Overview
This project uses Python and Selenium to scrape job listings from Wuzzuf website.  
The extracted data is cleaned and structured to analyze job market trends in Egypt for Analyst roles.

The goal is to transform raw job postings into structured data for better understanding of hiring trends, required skills, and market demand.

---

## 🎯 Objective
- Extract job listings from Wuzzuf
- Filter jobs based on location (Egypt, Cairo)
- Clean and structure scraped data
- Analyze job market trends

---

## 📌 Key Features
- Automated web scraping using Selenium
- Dynamic filtering (Country & City)
- Pagination handling across multiple pages
- Data cleaning and transformation
- Extraction of job details (company, experience, skills, etc.)

---

## 📊 Dataset Columns
- Job Name  
- Company Name  
- Location  
- Employment Type  
- Work Mode (Remote / Hybrid / On-site)  
- Experience  
- Skills  

---

## 🧹 Data Cleaning
- Removed unnecessary formatting characters from scraped text
- Standardized company names
- Extracted structured data from unstructured job descriptions
- Handled missing and inconsistent values

---

## 📈 Key Insights (Optional Future Analysis)
- Most in-demand skills in Analyst roles
- Top hiring companies in Egypt
- Experience level distribution
- Remote vs On-site job distribution
